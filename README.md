# Classacharm

Classacharm is a web platform designed for friends to share notes and important classroom messages. Whether you miss a class or lose track of important WhatsApp messages, Classacharm aims to provide a centralized solution for staying updated and informed.

## Features

- **Note Sharing**: Easily share and access notes from classes.
- **Message Board**: Post and view important classroom messages.
- **User Authentication**: Secure login and authentication system.
- **Mobile Responsive**: Designed to work seamlessly on desktop and mobile devices.

## Access Classacharm

You can access the live version of Classacharm [here](https://class-charm.vercel.app/).

## Installation

If you want to run Classacharm locally or contribute to its development, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/classacharm.git
   ```

2. Navigate to the project directory:

   ```bash
   cd classacharm
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm start
   ```

5. Open your browser and visit `http://localhost:3000` to see the app running locally.

## Technologies Used

- **Frontend**: React.js, Redux
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Authentication**: JSON Web Tokens (JWT)
- **Deployment**: Vercel (for frontend), Heroku (for backend)
